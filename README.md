Unique Tech Καβάλα
Παρουσίαση project

Είμαστε η ομάδα Unique Tech από την Καβάλα και έχουμε μια πρόταση η οποία θα βοηθήσει την κοινότητα και θα εξασφαλίσει ένα καλύτερο μέλλον για όλους μας. 
Παρατηρήσαμε ότι στις καθημερινές δραστηριότητες των γονιών μας αλλά και όλων των κατοίκων της Καβάλας γίνεται σπατάλη νερού σε πολύ μεγάλο βαθμό. Προσπαθήσαμε να μάθουμε περισσότερα για τη σπατάλη νερού και διαπιστώσαμε ότι σύμφωνα με έρευνες η σπατάλη είναι τεράστια σε αντίθεση με τις διαθέσιμες ποσότητες νερού οι οποίες είναι περιορισμένες. Ιδιαίτερα στην πόλη μας τα προβλήματα υδροδότησης είναι μεγάλα μιας και οι διαθέσιμες πηγές υδροληψίας μειώνονται συνεχώς ιδιαίτερα τα καλοκαίρια. 
Για το λόγο αυτό προτείνουμε ένα project το οποίο μας δείχνει τρόπους να εκμεταλλευτούμε νερό το οποίο θα είχαμε σπαταλήσει. Δημιουργήσαμε μια πρόταση η οποία εκμεταλλεύεται νερό που κανονικά θα είχε παταχθεί για τις ανάγκες ποτίσματος των δημοτικών κήπων της πόλης μας αλλά και για τη δημιουργία δημοτικού λαχανόκηπου ο οποίος θα έχει εκπαιδευτικό χαρακτήρα μιας και θα μπορούν να τον επισκέπτονται σχολεία και εκεί θα μαθαίνουν για τις βιολογικές καλλιέργειες και την εξοικονόμηση φυσικών πόρων. Το project μας εκμεταλλεύεται νερό σε ένα μεγάλο εύρος πηγών, από την εκμετάλλευση νερού που πετιέται από σπίτια (ακόμη και από αφυγραντήρες και κλιματιστικά) μέχρι το νερό της βροχής. Το project μας συλλέγει νερό από τις παραπάνω πηγές, το αποθηκεύει και το χρησιμοποιεί κατάλληλα. Σε όλα αυτά η τεχνολογία εξασφαλίζει την αποδοτικότερη χρήση μέσα από έξυπνα συστήματα συλλογής και διαχείρισης του νερού.
Θα παρουσιάσουμε το παραπάνω έργο μιας μακέτας που θα φτιάξουμε
